Эта программа предназначена для сортировки массивов числовых данных
Для начала, запустите файл runme.bat и введите строку "Perl TestSystem.pl"
Пользователь может выбрать следующие опции:
(Выберите нужный вариант)
1) Сортировка
Сортировка числовых данных, с помощью выбранного алгоритма
2) Время работы
Информация о времени в микросекундах определееной сортировки, в зависимости от числа элементов и сложности алгоритма
3) Перестановка
Число перестановок в зависимости от выбора алгоритма и количества элементов в массиве
4) Генерация данных для построения графика
Предлагается выбор, чтобы генерировать данные для графиков зависимости:
	1) Время от числа элементов
	2) Число перестановок из числа элементов
* Для каждого варианта можно сгенерировать данные как сразу для всех функций, так и для каждой функции отдельно
5) Информация о программе
Режим вывода информации на консоль или возможность открыть соответствующий файл
